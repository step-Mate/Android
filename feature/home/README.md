# Home Feature

Home 은 로그인을 하지 않아도 이용 가능한 걸음수 수집 및 시각화한 달력 과 차트를 제공해주는 화면입니다.

# Operations

- **걸음수 수집** : 사용자는 실시간으로 수집된 걸음수를 바탕으로 칼로리(kcal), 거리(km), 시간(분) 단위로 변환하여 시각화 된 차트(chart) 로 세분화 하여 볼 수 있다.
